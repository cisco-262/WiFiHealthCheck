# WiFi Health Check Website

Static marketing and support site for WiFi Health Check.

## Structure

- `index.html` - product landing page
- `privacy/index.html` - privacy policy
- `support/index.html` - support page
- `about/index.html` - about page
- `assets/css/styles.css` - shared styling
- `assets/js/main.js` - mobile menu behavior
- `assets/images/` - app icon and product images
- `robots.txt` and `sitemap.xml` - SEO support

## Notes

The current canonical domain is set to `https://wifi.opshome.run/`.

The App Store download button uses `#download` as a placeholder until the final App Store URL is available.
