About WiFi Health Check

WiFi Health Check is a focused iPhone app for quick WiFi and LAN troubleshooting, built by OpsHome.

What WiFi Health Check is

WiFi Health Check helps users understand common WiFi issues from their iPhone. It checks public internet connectivity, current WiFi context such as SSID, BSSID, and security type, local gateway context, custom LAN TCP reachability, and WiFi roaming behavior.

Why it exists

WiFi problems are often hard to describe. A simple local report can show whether the issue is external connectivity, a LAN endpoint, gateway configuration, WiFi security context, or roaming between access points.

Part of the OpsHome product family

OpsHome builds practical tools for operations visibility. WiFi Health Check covers foreground local diagnosis. OpsHome NOC covers broader uptime monitoring, private services, Docker Probe, Synology, Proxmox, Linux hosts, Docker containers, alerts, and historical views.
