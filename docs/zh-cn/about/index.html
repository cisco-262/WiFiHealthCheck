关于 WiFi Health Check

WiFi Health Check 是 OpsHome 构建的一款专注 WiFi 和局域网快速排障的 iPhone 应用。

它是什么

WiFi Health Check 帮助用户从 iPhone 理解常见 WiFi 问题。它检查公网连通性、当前 WiFi 上下文如 SSID、BSSID 和安全类型、网关上下文、自定义 LAN TCP 可达性，以及 WiFi 漫游行为。

为什么需要它

WiFi 问题往往很难描述。一份本地报告可以帮助判断问题来自外网连通性、局域网设备、网关配置、WiFi 安全上下文，还是接入点之间的漫游。

OpsHome 产品体系

OpsHome 构建面向实际运维可见性的工具。WiFi Health Check 覆盖前台本地诊断；OpsHome NOC 覆盖更完整的正常运行时间监控、私有服务、Docker Probe、Synology、Proxmox、Linux 主机、Docker 容器、告警和历史视图。
