面向真实 WiFi 排障的专注工具

WiFi Health Check 是前台本地诊断工具，不是后台监控或抓包工具。

1

一键网络诊断

在一份报告中检查公网网站连通性、DNS 行为、本机 IP、WiFi 安全类型、网关地址和网关 TCP 探测上下文。

T

局域网 TCP 稳定性

对路由器、NAS、打印机、Proxmox 主机或 AP 管理地址进行自定义 TCP 端口连续测试。

R

WiFi 漫游报告

移动过程中记录 SSID 和 BSSID 样本，查看从一个 WiFi 节点到另一个节点的路径。

W

WiFi 安全类型

当 iOS 提供时显示当前 WiFi 的安全类型，包括开放、WEP、个人和企业网络。

G

网关信息

显示当前网关地址作为辅助信息，不把网关端口关闭直接判定为互联网故障。

S

可分享文本报告

诊断、稳定性和漫游结果保存在本机，需要求助时再主动分享。

工作方式

连接 WiFi 后打开应用

请使用真实 iPhone。iOS 可能要求位置权限才能提供 SSID、BSSID 和安全类型。

选择合适测试

根据问题选择一键诊断、LAN TCP 稳定性或漫游测试。

查看本地结果

理解公网连通性、DNS、网关、TCP 超时和漫游路径变化。

需要时再分享

报告默认留在设备上，只有你通过 iOS 分享时才会离开设备。

区分互联网、局域网和漫游问题

WiFi 问题可能来自宽带、DNS、路由器 WAN、局域网设备、弱信号或客户端黏在错误 AP 上。WiFi Health Check 把这些信号拆成可比较的小报告。
